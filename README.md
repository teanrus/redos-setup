# 🚀 Автоматическая настройка РЕД ОС 7.3

[![Version](https://img.shields.io/badge/version-2.7-green.svg)](https://github.com/teanrus/redos-setup/releases)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-RED%20OS%207.3-red.svg)](https://redos.ru/)

Скрипт для выборочной автоматической установки и настройки программного обеспечения на **РЕД ОС 7.3**. Позволяет установить только нужные программы, скачивая их из GitHub Releases по мере необходимости.

---

### 📋 Быстрая установка

```bash
curl -sL curl -sL https://github.com/teanrus/redos-setup/releases/download/v2.7/setup.sh | bash | sudo bash
```

#### Или с сохранением файла:
```bash
wget curl -sL https://github.com/teanrus/redos-setup/releases/download/v2.7/setup.sh | bash
chmod +x setup.sh
sudo ./setup.sh
```
---

## 🎯 Что делает скрипт
Скрипт последовательно запрашивает у пользователя согласие на установку каждой программы и скачивает только выбранные компоненты. Это позволяет:
- Экономить трафик (скачивается только нужное)
- Экономить дисковое пространство (файлы удаляются после установки)
- Гибко настраивать систему под конкретные задачи

### Базовые настройки системы (выполняются всегда)
| Действие                    | Описание                                                 |
| :-------------------------- | :------------------------------------------------------- |
| Отключение SELinux          | Отключает SELinux для совместимости с корпоративным ПО   |
| Настройка DNF               | Увеличивает скорость загрузки до 10 параллельных потоков |
| Создание рабочей директории | Создает /home/inst для временного хранения файлов        |

### 🔄 Как это работает
Процесс установки каждой программы:
- апрос согласия — скрипт спрашивает: "Установить [программу]? (y/n)"
- Скачивание — при согласии файл загружается из GitHub Releases
- Установка — пакет устанавливается через dnf или специальные скрипты
- Очистка — загруженный файл удаляется из /home/inst

#### Пример для ViPNet (с дополнительным выбором):
```bash
Установить ViPNet? (y/n): y
=== Выбор версии ViPNet ===
1. ViPNet Client (без деловой почты)
2. ViPNet + Деловая почта (DP)
Выберите вариант (1 или 2): 1
Загрузка vipnetclient-gui_gost_ru_x86-64_4.15.0-26717.rpm...
✓ Установка ViPNet Client успешно выполнено
```

### 🔧 Особенности установки отдельных программ
Telegram
- Распаковывается в /opt/telegram
- Создается символическая ссылка /usr/bin/telegram
- Добавляется ярлык в меню приложений

Шрифты Liberation
- станавливаются в /usr/share/fonts/liberation
- После установки выполняется fc-cache -fv для обновления кэша шрифтов

КриптоПро
- начала устанавливаются зависимые пакеты: ifd-rutokens, token-manager, gostcryptogui, caja-gostcryptogui
- Затем распаковывается архив kriptopror4.tar.gz
- Запускается графический установщик install_gui.sh

ViPNet
- Client версия: один RPM-пакет
- DP версия: архив с двумя RPM-пакетами, устанавливаются оба

### 🐛 Устранение неполадок
Проблема: "Permission denied"

Решение: Запустите скрипт с правами root:

```bash
sudo ./setup.sh
```
Проблема: Не удается скачать файл

Решение: Проверьте интернет-соединение и доступность GitHub:

```bash
curl -I https://github.com/teanrus/redos-setup/releases/download/v1.0/setup.sh
```
Проблема: Ошибка при установке RPM

Решение: Убедитесь, что система обновлена:

```bash
dnf update -y
```

Проблема: Нет доступа к GitHub

Решение: Используйте локальный источник или настройте прокси:

```bash
export http_proxy=http://proxy.example.com:8080
```
