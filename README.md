## 📖 О проекте

Интеллектуальный bash-скрипт для выборочной установки и настройки программного обеспечения на **РЕД ОС 7.3**.

Скрипт последовательно запрашивает согласие на установку каждого компонента и **скачивает только выбранные программы** из GitHub Releases, экономя трафик и время.

### ✨ Особенности

- 🎯 **Выборочная установка** — устанавливается только то, что нужно
- 💾 **Экономия трафика** — файлы скачиваются только при подтверждении
- 🗑️ **Автоочистка** — временные файлы удаляются после установки
- 🎨 **Цветной вывод** — понятная индикация статуса
- 🔄 **Ленивая загрузка** — скачивание только при согласии

## 🚀 Быстрая установка

```bash
# Вариант 1 (рекомендуемый)
curl -sL https://github.com/teanrus/redos-setup/releases/latest/download/setup.sh | sudo bash

# Вариант 2 (с сохранением файла)
wget https://github.com/teanrus/redos-setup/releases/latest/download/setup.sh
chmod +x setup.sh
sudo ./setup.sh

# Вариант 3 (если нет wget)
curl -L -o setup.sh https://github.com/teanrus/redos-setup/releases/latest/download/setup.sh
chmod +x setup.sh
sudo ./setup.sh
```

## 🎮 Использование

### Интерактивный режим

```bash
sudo ./setup.sh
```

Скрипт задаст вопросы:

```text
Установить базовую систему? (y/n): y
Установить шрифты Liberation? (y/n): y
Установить браузер Chromium-GOST? (y/n): n
...
```

### Неинтерактивный режим (CLI)

```bash
# Установка конкретных компонентов
sudo ./setup.sh --install telegram

# Установка нескольких компонентов
sudo ./setup.sh --install telegram,vipnet,1c

# С автоматическим подтверждением
sudo ./setup.sh --install telegram,vipnet --yes

# Показать список компонентов
./setup.sh --list

# Показать справку
./setup.sh --help
```

### Доступные компоненты для --install

| Компонент | Описание               |
| :-------- | :--------------------- |
| telegram  | Telegram               |
