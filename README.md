**Интеллектуальный скрипт для выборочной установки и настройки программного обеспечения на РЕД ОС 7.3.**

Скрипт последовательно запрашивает согласие на установку каждого компонента и **скачивает только выбранные программы** из GitHub Releases, экономя трафик и время. Поддерживает ленивую загрузку — файлы загружаются только при подтверждении пользователя и удаляются после установки.

---

#### 📋 Быстрая установка

```bash
curl -sL https://github.com/teanrus/redos-setup/releases/download/v2.7/setup.sh | sudo bash
```

#### Или с сохранением файла:
```bash
wget curl -sL https://github.com/teanrus/redos-setup/releases/download/v2.7/setup.sh
chmod +x setup.sh
sudo ./setup.sh
```
---

## 🎯 Что делает скрипт
Скрипт последовательно запрашивает у пользователя согласие на установку каждой программы и скачивает только выбранные компоненты. Это позволяет:
- Экономить трафик (скачивается только нужное)
- Экономить дисковое пространство (файлы удаляются после установки)
- Гибко настраивать систему под конкретные задачи

### 🔄 Как это работает
Принцип ленивой загрузки:
- Запрос согласия — скрипт спрашивает: "Установить [программу]? (y/n)"
- Скачивание — при согласии файл загружается из GitHub Releases
- Установка — пакет устанавливается через dnf или специальные скрипты
- Очистка — загруженный файл удаляется из /home/inst

#### Пример для ViPNet (с дополнительным выбором):
```text
Установить ViPNet? (y/n): y
=== Выбор версии ViPNet ===
1. ViPNet Client (без деловой почты)
2. ViPNet + Деловая почта (DP)
Выберите вариант (1 или 2): 1
Загрузка vipnetclient-gui_gost_ru_x86-64_4.15.0-26717.rpm...
✓ Установка ViPNet Client успешно выполнено
```

### 🔧 Особенности установки отдельных программ
Telegram
- Распаковывается в /opt/telegram
- Создается символическая ссылка /usr/bin/telegram
- Добавляется ярлык в меню приложений

Шрифты Liberation
- Устанавливаются в /usr/share/fonts/liberation
- После установки выполняется fc-cache -fv для обновления кэша шрифтов

КриптоПро
- начала устанавливаются зависимые пакеты: ifd-rutokens, token-manager, gostcryptogui, caja-gostcryptogui
- Затем распаковывается архив kriptopror4.tar.gz
- Запускается графический установщик install_gui.sh

ViPNet
- Client версия: один RPM-пакет
- DP версия: архив с двумя RPM-пакетами, устанавливаются оба
