# Описание
Скрипт для полной автоматической настройки операционной системы **РЕД ОС 7.3** после установки.
Позволяет за одну команду установить и настроить всё необходимое для работы.

## Быстрая установка
### Скачать и запустить скрипт
```bash
curl -L -O https://raw.githubusercontent.com/teanrus/redos-setup/main/setup.sh
chmod +x setup.sh
sudo ./setup.sh
```
## Что делает скрипт
### 1. 🔒 Безопасность и системные настройки
- Отключает SELinux для совместимости с некоторым корпоративным ПО
- Настройка DNF Увеличивает скорость загрузки пакетов до 10 параллельных потоков
- Настройка TRIM Включает автоматическую оптимизацию SSD-накопителей

### 2. 📦 Установка репозиториев
- r7-release - Репозиторий R7 Office для установки офисного пакета
- yandex-browser-release - Репозиторий Яндекс.Браузера
- MAX - Репозиторий MAX Desktop

### 3. ⚙️ Обновление системы
- Обновление всех установленных пакетов до актуальных версий
- Установка последнего ядра из репозитория redos-kernels6
- Обновление конфигурации GRUB

### 4. 💻 Установка основного ПО
- pavucontrol - Управление звуком (микшер PulseAudio)
- r7-office - Офисный пакет R7 Office (аналог Microsoft Office)
- yandex-browser-stable - Яндекс.Браузер для организаций
- sshfs - Монтирование удаленных директорий через SSH
- pinta - Графический редактор (аналог Paint.NET)
- perl-Getopt-Long - Модуль Perl для работы Kaspersky
- perl-File-Copy - Модуль Perl для работы Kaspersky

### 5. 🌐 Настройка ViPNet (опционально)
- Замена DNS-серверов на корпоративные (10.13.60.2, 10.14.100.222)
- Включение параметра iptables=off для корректной работы ViPNet
- Сохранение конфигурации в /etc/vipnet.conf

### 6. 🔐 Установка КриптоПро (опционально)
- ifd-rutokens - Драйвер для работы с Рутокен (токены электронной подписи)
- token-manager - Менеджер токенов для управления электронными ключами
- gostcryptogui - GUI для работы с ГОСТ-шифрованием
- caja-gostcryptogui - Интеграция ГОСТ-шифрования в файловый менеджер Caja
- КриптоПро CSP - Установка из локального репозитория (папка R4)

### 7. 🛡️ Установка Kaspersky Agent (опционально)
- Установка агента для управления через Kaspersky Security Center
- Файл: klnagent64-15.4.0-8952.x86_64.sh

### 8. 📊 Установка 1С:Предприятие (опционально)
- Загрузка и установка платформы 1С:Предприятие версии 8.3.24.1691
- Применение фиксов для корректной работы в РЕД ОС
- Автоматическая очистка временных файлов после установки

### 9. 🖥️ Настройка моноблока KSG (опционально)
- Добавление команды xrandr --output HDMI-3 --primary в /etc/gdm/Init/Default
- Настройка основного монитора для моноблоков KSG (исправление отображения)
- Команда вставляется перед exit 0 для гарантированного выполнения

### 10.🧹 Завершающие действия
- Очистка всех временных файлов и RPM-пакетов
- Включение автоматической оптимизации SSD
- Обновление конфигурации GRUB
- Запрос на перезагрузку системы

## 🎯 Выбор источника загрузки
- При запуске скрипт предлагает выбрать источник для загрузки пакетов:

### 1. 🌍 GitHub (рекомендуется)
Преимущества:
- Загрузка из интернета без необходимости в VPN
- Автоматическая проверка обновлений скрипта
- Контрольные суммы для проверки целостности файлов
- Доступно из любой точки мира
### 2. 🏢 Локальная сеть (SMB)
Преимущества:
- Высокая скорость загрузки внутри локальной сети
- Только для локальной сети Департамента образования Приуральского района

## ✅ Подтверждение установки компонентов
Скрипт запрашивает подтверждение перед установкой каждого компонента:

- Установить ViPNet? (y/n): y
- Установить КриптоПро и дополнительные пакеты? (y/n): y
- Установить Kaspersky Agent? (y/n): n
- Установить 1С? (y/n): y
- Настраиваем моноблок KSG? (y/n): n
- Это позволяет гибко настраивать систему под конкретные потребности.
