# redos-setup
# Репозиторий для автоматической настройки РЕД ОС 7.3

## Описание
Скрипты и пакеты для автоматической установки и настройки РЕД ОС 7.3.

## Быстрая установка
```bash
# Скачать и запустить скрипт
curl -L -O https://raw.githubusercontent.com/teanrus/redos-setup/main/setup.sh
chmod +x setup.sh
sudo ./setup.sh
