# redos-setup
# Репозиторий для автоматической настройки РЕД ОС 7.3

## Описание
Скрипты и пакеты для автоматической установки и настройки РЕД ОС 7.3.

## Быстрая установка
```bash
# Скачать и запустить скрипт
curl -L -O https://github.com/ВАШ_НИК/redos-setup/releases/latest/download/setup.sh
chmod +x setup.sh
sudo ./setup.sh
