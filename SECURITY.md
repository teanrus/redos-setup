# Security Policy

## Supported Versions

С точки зрения сопровождаемой логики проекта:

- РЕД ОС `7.x`: полный сценарий установки;
- РЕД ОС `8+`: частичная поддержка, без установки `КриптоПро` и `ViPNet` из текущего релиза.

## Reporting a Vulnerability

Если вы обнаружили проблему безопасности, не публикуйте детали сразу в открытом issue, если уязвимость:

- позволяет выполнить нежелательные действия с правами `root`;
- приводит к загрузке или запуску недоверенных артефактов;
- затрагивает обработку системных конфигурационных файлов;
- позволяет нарушить ожидаемые ограничения совместимости.

Для сообщения об уязвимости используйте контакт:

- [tyanrv@lbt.yanao.ru](mailto:tyanrv@lbt.yanao.ru)

В сообщении желательно указать:

- описание проблемы;
- затронутую версию или релиз;
- условия воспроизведения;
- ожидаемое и фактическое поведение;
- оценку влияния на систему.

## Operational Notes

Проект выполняется с повышенными привилегиями и изменяет системную конфигурацию. Перед применением изменений рекомендуется проверять релизные артефакты, тестировать сценарий на стенде и использовать резервные копии для рабочих систем.
