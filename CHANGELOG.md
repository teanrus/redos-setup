# Changelog

Все заметные изменения в проекте `redos-setup` рекомендуется отражать в этом файле.

Формат основан на принципах Keep a Changelog. Семантическое версионирование может использоваться по мере развития проекта, но записи в журнале должны быть понятны пользователю независимо от выбранной схемы нумерации.

## [Unreleased]

### Changed

- Добавлено определение версии ОС через `/etc/os-release`.
- Добавлена логика совместимости для РЕД ОС `7.x` и `8+`.
- На РЕД ОС `8+` установка `КриптоПро` и `ViPNet` из текущего релиза автоматически исключается как несовместимая.
- Обновлён `README.md` под более зрелый open-source стиль без потери пользовательской информации.
- Добавлены проектные документы `CONTRIBUTING.md`, `CHANGELOG.md` и `SECURITY.md`.

## [2.8]

### Added

- Поддержка ленивой загрузки компонентов из GitHub Releases.
- Выборочная установка программ по подтверждению пользователя.
- Базовый набор системных и прикладных компонентов для настройки рабочего места.

### Notes

- Исторически скрипт разрабатывался и тестировался под РЕД ОС `7.3`.
